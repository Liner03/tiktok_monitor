# dog_love
主要功能：24小时检测女神动态，这里抖音为例！

1:安装python3最新版  

2:修改dog_love.py程序里的部分信息为自己的！ 

3:将此压缩包解压到桌面，直接运行点击「 舔狗.bat 」直接运行即可！  

4:如果程序报错，请安装dog_love.py里面所用到的库。  

5:如果想让程序24小时不断运行，请将此脚本添加为计划任务  

6:计划任务如何使用，请自行百度
